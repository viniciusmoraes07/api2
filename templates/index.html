Welcome!

API created to validate zip code numbers on the Brazil post office website.


Enter the zip code you want to validate and a json will be returned with the information:

  • If the zip code is valid it returns the following information: Street / Neighborhood / District; Locality / State.
  • For invalid zip code returns a message like the content: "Invalid zip code".
